# AdSafe PRD (Product Requirements Document)

> 의료광고 규제 리스크 검수 및 교육 서비스

**버전**: MVP 1.0 | **최종 수정**: 2026-02-05

---

## 1. 제품 개요

### 1.1 한 줄 정의
의료광고 담당자가 규제 위반 없이 안전하게 광고를 제작할 수 있도록 돕는 **자동 검수 + 퀴즈 교육** 서비스

### 1.2 핵심 기능

| 모드 | 기능 | 대상 |
|------|------|------|
| **AdSafe** | 광고 문구 자동 검수, 리스크 탐지, 수정 가이드 | 마케터 |
| **AduSafe** | 퀴즈 기반 규제 학습, 오답 복습 | 신입/학습자 |
| **Admin** | 리스크 유형, 룰, 사용자 관리 | 관리자 |

---

## 2. 문제 정의

### 2.1 현재 문제

```
의료광고 담당자 → 복잡한 규제 → 수동 검수 → 위반 리스크
                     ↓
              • 검수 시간 과다 (건당 45분)
              • 휴먼 에러로 위반
              • 신입 교육 체계 부재
```

### 2.2 문제의 영향

| 영향 | 내용 |
|------|------|
| 금전적 | 과태료 50~300만원, 영업정지 |
| 시간적 | 월 37시간 이상 검수에 소요 |
| 품질적 | 담당자별 검수 기준 편차 |

### 2.3 해결 방안

| 문제 | 해결책 |
|------|--------|
| 규제 복잡성 | 룰 기반 자동 탐지 |
| 검수 비효율 | 즉시 결과 + 수정 가이드 |
| 교육 부재 | 퀴즈 + 복습 시스템 |

## 6. 서비스 범위

### 6.1 포함 (In Scope)
- 텍스트 기반 광고 문구 검수
- 키워드/정규식 룰 기반 탐지
- 퀴즈 기반 교육
- 기본 사용자/룰 관리

### 6.2 제외 (Out of Scope)
- 이미지/영상 검수
- 법적 심의 대행 (공식 심의 기관 아님)
- 다국어 지원
- 의료 외 업종

### 6.3 면책 사항
> AdSafe는 참고용 검수 도구이며, 검수 결과에 법적 효력이 없습니다. 최종 판단은 사용자 책임입니다.
