# AdSafe 사용 가이드

**버전**: MVP 1.0 | **최종 수정**: 2026-02-05

---

## 1. 서비스 소개

AdSafe는 의료광고 문구의 규제 리스크를 **자동 검수**하고, **퀴즈로 학습**할 수 있는 서비스입니다.

| 기능 | 설명 |
|------|------|
| **검수하기** | 광고 문구 입력 → 위반 리스크 탐지 → 수정 가이드 |
| **학습하기** | 퀴즈 10문제 → 해설 확인 → 오답 복습 |

---

## 2. 시작하기

### 2.1 회원가입

1. 메인 페이지에서 **회원가입** 클릭
2. 이메일, 비밀번호(6자 이상), 이름 입력
3. **가입하기** 클릭
4. 로그인 페이지로 이동

### 2.2 로그인

1. 이메일, 비밀번호 입력
2. **로그인** 클릭
3. 메인 대시보드로 이동

---

## 3. 검수하기 (AdSafe Mode)

### 3.1 광고 문구 검수

1. **검수하기** 메뉴 클릭
2. (선택) 광고 유형, 제목 입력
3. 광고 문구 입력
4. **검수하기** 버튼 클릭

### 3.2 검수 결과 확인

```
┌─────────────────────────────────────┐
│  요약 위험도: 높음                   │
│  적발 건수: 2건                      │
├─────────────────────────────────────┤
│  [적발 1] RISK_GUARANTEE            │
│  • 매칭: "100% 완치"                │
│  • 위험도: 높음                      │
│  • 설명: 치료 효과 보장 표현 금지    │
│  • 수정: 보장 표현 삭제              │
├─────────────────────────────────────┤
│  [적발 2] RISK_PRICE_EXCESSIVE      │
│  • 매칭: "50% 할인"                 │
│  • 위험도: 보통                      │
│  • 설명: 과도한 할인 표현            │
│  • 수정: 조건 명시                   │
└─────────────────────────────────────┘
```

### 3.3 검수 이력 조회

1. **검수 이력** 메뉴 클릭
2. 필터 설정 (광고 유형, 위험도, 기간)
3. 목록에서 **상세보기** 클릭

---

## 4. 학습하기 (AduSafe Mode)

### 4.1 퀴즈 학습

1. **학습하기** 메뉴 클릭
2. **학습 시작** 클릭
3. 10문제 풀기 (4지선다)
4. 결과 및 해설 확인

### 4.2 복습하기

1. **복습하기** 메뉴 클릭
2. 틀린 문제 개수 확인
3. **복습 시작** 클릭
4. 틀린 문제만 다시 풀기

### 4.3 학습 현황

**학습 현황** 페이지에서 확인:
- 총 퀴즈 횟수
- 총 문제 수 / 정답 수
- 평균 정답률
- 최근 학습 이력

---

## 5. 계정 관리

### 5.1 계정 정보 확인
- 상단 메뉴 **계정** 클릭
- 이메일, 이름 확인

### 5.2 회원탈퇴
1. 계정 페이지에서 **회원탈퇴** 클릭
2. 비밀번호 입력
3. **탈퇴하기** 클릭

### 5.3 로그아웃
- 상단 메뉴 **로그아웃** 클릭

---

## 6. 관리자 기능 (Admin 전용)

> Admin 권한 사용자만 접근 가능

### 6.1 리스크 유형 관리

**메뉴**: 어드민 콘솔 → 리스크 유형 관리

| 작업 | 방법 |
|------|------|
| 추가 | **리스크 유형 추가** 클릭 → 정보 입력 → 저장 |
| 수정 | 목록에서 **수정** 클릭 → 정보 수정 → 저장 |
| 삭제 | 목록에서 **삭제** 클릭 (연결된 룰 있으면 삭제 불가) |

**입력 항목**:
- 리스크 코드: `RISK_XXX_YYY` 형식
- 대분류, 중분류, 판정단위
- 기본 위험도: 낮음/보통/높음

### 6.2 룰 관리

**메뉴**: 어드민 콘솔 → 룰 관리

| 작업 | 방법 |
|------|------|
| 추가 | **룰 추가** 클릭 → 정보 입력 → 저장 |
| 수정 | 목록에서 **수정** 클릭 |
| 삭제 | 목록에서 **삭제** 클릭 |

**입력 항목**:
- 룰명, 리스크 코드
- 룰 유형: keyword (쉼표 구분) / regex (정규식)
- 패턴, 위험도, 설명

### 6.3 룰셋 버전 관리

**메뉴**: 어드민 콘솔 → 룰셋 버전

| 작업 | 방법 |
|------|------|
| 버전 생성 | **새 버전** 클릭 → 버전명 입력 → 저장 |
| 활성화 | **활성화** 클릭 (기존 활성 버전 자동 비활성화) |
| 비활성화 | **비활성화** 클릭 |
| 삭제 | **삭제** 클릭 (활성 버전 삭제 불가) |

### 6.4 사용자 관리

**메뉴**: 어드민 콘솔 → 사용자 관리

| 작업 | 방법 |
|------|------|
| 권한 변경 | **수정** 클릭 → 역할 변경 → 저장 |
| 삭제 | **삭제** 클릭 |

## 8. FAQ

### Q: 검수 결과를 믿어도 되나요?
> AdSafe는 참고용 도구입니다. 최종 판단은 사용자 책임이며, 필요시 공식 심의 기관에 문의하세요.

### Q: 이미지 광고도 검수할 수 있나요?
> 현재 텍스트만 지원합니다. 이미지 검수는 향후 업데이트 예정입니다.

### Q: 복습할 문제가 없다고 나와요
> 아직 틀린 문제가 없습니다. 먼저 **학습하기**에서 퀴즈를 풀어보세요.

### Q: 비밀번호를 잊어버렸어요
> 현재 비밀번호 재설정 기능은 미지원입니다. 관리자에게 문의하세요.
